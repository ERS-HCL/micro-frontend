# Microfrontend

Container folder for two different microfrontend approaches : 
1) Runtime Integration via web components , it supports multiple UI frameworks for example React and AngularJS 
   with a React Application Shell 
2) Runtime Integration via Javascript: Have the ability to have microfrontends run independently as well part of an application shell (using React)
